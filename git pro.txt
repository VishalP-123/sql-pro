git init ===> convert your normal directory into git directory. Initialize the git for the project

git add <path>
git add --all ===> add the files indicated in the staging area

git commit -m "done" ===> commit the change as snapshot in the git

git remote add petstore https://github.com/ekitv2021/petstore_7_9.git ===> create a remote refer named petstore to the remote git

git push petstore main ===> push the changes from the local git to the remote git
